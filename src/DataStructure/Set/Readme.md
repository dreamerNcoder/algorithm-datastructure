# Set Datastructure

A set is a collection of distinct objects


| Name                   | Time Complexity |
|------------------------|-----------------|
| quick-find             | O(m n)          |
| quick-union            | O(m n)          |
| weighted QU            | O(n + m log2n)  |
| QU + path comp         | O(n + m log2n)  |
| weighted QU + path com | O(n + m log*n)  |
